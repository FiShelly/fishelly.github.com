命令行参数

还可以使用以下命令行标志进一步配置quick-project-cli。

Usage: qp-cli [options] [dir]

Options:

  -V, --version  output the version number
  --vue          add vue.js support
  --react        add react support
  --less         add less support(default)
  --sass         add less support(default is less)
  --git          add .gitignore
  -h, --help     output usage information

其他例子

创建一个支持vue的项目

参数解析：

  • –vue 就是为项目添加vue支持，如果去掉，则变成普通的es6项目。

  • 默认情况下是支持less的，但是如果加上参数–sass 则变成支持sass。

进入到项目目录后，按照正常那样安装依赖后，执行npm run dev即可启动项目，在浏览器地址栏输入：http://localhost:4000,则进入到HelloWorld页面，如果4000端口被占用，则在server.js修改端口即可。
